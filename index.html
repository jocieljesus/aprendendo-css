Aprendendo CSS

O que é o CSS?

É a linguagem usada para definir a apresentação visual de páginas web escritas em HTML, controlando cores, layout, fontes e responsividade. Ele separa o conteúdo da estrutura, facilitando a estilização e manutenção de sites, permitindo que a mesma estrutura HTML receba diferentes aparências.

Funcionalidades principais

  • Layout: Controle de posicionamento usando Flexbox e Grid.
  • Responsividade: Uso de media queries para adaptar o layout a diferentes tamanhos de tela.
  • Estilização: Definição de cores, fontes, bordas, espaçamentos (padding/margin) e tamanhos.
